1. [x] ランナーマシン上で GitHub 提供のスクリプトを使用する

1. [ ] 事前定義された GitHub Actions ワークフロー `network-connectivity.yml` を使用する
1. [ ] ランナーアプリケーションがランナーマシンにインストールされると、GitHub がネットワーク接続を自動的に検証します
