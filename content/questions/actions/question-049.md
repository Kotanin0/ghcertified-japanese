- [x] GitHub はさまざまなカテゴリ、言語、ツール向けにスターターワークフローを提供および管理している
- [x] 組織は、組織内のユーザー向けにカスタムスターターワークフローを作成できる
- [ ] スターターワークフローは再利用可能なワークフローを呼び出すことができない
- [ ] スターターワークフローは GitHub の有料機能である
- [ ] スターターワークフローはそのまま使用可能で、変更や拡張はできない
