> https://docs.github.com/en/actions/security-guides/using-secrets-in-github-actions#naming-your-secrets
- [x] 同じ名前 `API_KEY` のリポジトリシークレットを作成する
- [x] 同じ名前 `API_KEY` の環境シークレットを作成する
- [ ] 同じ名前 `API_KEY` のエンタープライズシークレットを作成する
- [ ] 名前 `OVERRIDE_API_KEY` のエンタープライズシークレットを作成する
- [ ] 名前 `OVERRIDE_API_KEY` のリポジトリシークレットを作成する
- [ ] 名前 `OVERRIDE_API_KEY` の環境シークレットを作成する
- [ ] 名前 `REPOSITORY_API_KEY` のリポジトリシークレットを作成する
- [ ] 名前 `ENVIRONMENT_API_KEY` の環境シークレットを作成する
