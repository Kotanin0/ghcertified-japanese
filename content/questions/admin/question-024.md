- [x] 個人アクセストークン（PAT）
- [x] SSHキー
- [x] エンタープライズアカウント向けのSAML SSO
- [x] パスキー認証（パスワードレスログインのオプトインベータ版）
- [x] サードパーティアプリ統合用のOAuthトークン
- [ ] モバイルデバイスにリンクされたQRコードシステム
- [ ] FacebookやGoogleなどのソーシャルメディアアカウントを使用した認証
