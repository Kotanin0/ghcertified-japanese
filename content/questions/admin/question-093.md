> https://docs.github.com/en/packages/learn-github-packages/introduction-to-github-packages#authenticating-to-github-packages
1. [x] 適切なスコープを持つPATを使用して認証する。
1. [ ] GitHub Packagesは認証なしで公開アクセスおよび共有が可能です。
1. [ ] GitHub PackagesへのアクセスはGitHub Enterpriseアカウントに限定されています。
1. [ ] 特別な認証トークンを必要とせず、デフォルトのGitHubログインを使用してアクセスします。
