archetype: "questions"
title: "Question 094"
---

> https://docs.github.com/en/packages/learn-github-packages/introduction-to-github-packages#authenticating-to-github-packages
1. [x] ワークフローリポジトリに関連付けられたパッケージを公開するには`GITHUB_TOKEN`を使用し、プライベートリポジトリからパッケージをインストールするにはPATを使用する。
1. [ ] GitHub PackagesはGitHub ActionsワークフローやCI/CDツールで使用できません。
1. [ ] GitHub Packagesをワークフローで使用するにはSSHキーのみが必要です。
1. [ ] 各ワークフロー実行ごとにパッケージを手動でダウンロードおよびアップロードする必要があります。
