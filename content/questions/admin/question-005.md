---
question: "GitHub Enterprise Server の診断ファイルを生成して共有する正しい方法はどれですか？（2 つ選択してください）"
archetype: "questions"
title: "質問 005"
---

> https://docs.github.com/en/enterprise-server/support/contacting-github-support/providing-data-to-github-support#creating-and-sharing-diagnostic-files
- [x] `ghe-diagnostics` コマンドラインユーティリティを使用してインスタンスの診断情報を取得します。
- [x] 管理コンソールのサポートタブに移動し、`診断情報をダウンロード` をクリックして診断ファイルを作成します。
- [ ] `ghe-support-bundle` コマンドを使用して診断ファイルを作成してアップロードします。
- [ ] GitHub モバイルアプリを使用して診断ファイルを生成します。
- [ ] `ghe-diagnostics -u` コマンドを実行して診断ファイルを作成し、自動的に GitHub サポートにアップロードします。
