1. [x] GitHub Actionsの使用は公開リポジトリでは無料です。
1. [ ] GitHub Actionsは公開リポジトリで使用された分数ごとに料金が発生します。
1. [ ] 公開リポジトリには、プライベートリポジトリと比較して半分の無料分数が与えられます。
1. [ ] GitHub Actionsは公開リポジトリでは利用できません。
