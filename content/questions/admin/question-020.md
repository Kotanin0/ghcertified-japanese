question: "エンタープライズ管理ユーザーのユーザー名とプロフィール情報に関して正しい記述はどれですか？"
archetype: "questions"
title: "Question 020"
---

> https://docs.github.com/en/enterprise-cloud@latest/admin/identity-and-access-management/understanding-iam-for-enterprises/about-enterprise-managed-users
1. [x] ユーザー名とプロフィール情報はエンタープライズの IdP を通じて設定され、ユーザー自身で変更することはできません。
1. [ ] ユーザーは GitHub 上でいつでもユーザー名とプロフィール情報を変更できます。
1. [ ] ユーザー名とプロフィール情報は、GitHub アカウントのセットアップ時にユーザー自身が設定します。
1. [ ] GitHub はセキュリティを強化するためにランダムなユーザー名とプロフィール情報を自動的に割り当てます。
