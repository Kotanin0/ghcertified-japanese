---
question: "組織に GitHub Marketplace のアプリをインストールする手順は何ですか？"
archetype: "questions"
title: "質問 008"
---

> https://docs.github.com/en/apps/using-github-apps/installing-a-github-app-from-github-marketplace-for-your-organizations
1. [x] GitHub Marketplace を閲覧し、アプリを選択してプランを選び、組織を選択してアプリを確認してインストールします。
1. [ ] GitHub サポートに連絡してアプリを手動でインストールし、メールの指示に従って設定します。
1. [ ] アプリを外部サイトからダウンロードし、手動で GitHub 組織のリポジトリにアップロードします。
1. [ ] 組織のダッシュボードから直接アプリをインストールし、特定のプランを選択する必要はありません。
